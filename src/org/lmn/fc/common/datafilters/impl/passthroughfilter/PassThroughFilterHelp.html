Pass Through Filter

The Pass Through Filter copies the RawData directly to the ProcessedData, without any modifications.

Mathematically, for each dataset type;

Timestamped
   (t, y) --> (t, y)

Indexed
   (x, y) --> (x, y)

This filter is set as the default choice, because in most cases you will probably want to examine the RawData as they were recorded, at least initially. When you know what processing is necessary, simply re-import using a different filter. Alternatively, use this command:

   DataProcessor.applyFilter()

which will allow repeated transformations without the need to import the data again. You may also transform the ProcessedData repeatedly, without re-reading the RawData.
