Starbase Logger


Look in workspace/staribus-stub for 'stub' XML files for testing the Staribus Plug and Play system.
They may be written to the 8 EEPROMs in a full system,
i.e. Controller, PrimaryPlugin and Plugins 2-7
using the setModuleConfiguration(module_id, filename) Command
The getConfiguration() Command will read all EEPROMs
and assemble the individual files into one XML document.
The console window will give a message to show if the document is validated against the schema. You may also use the DiscoveryTester Instrument in conjunction with a stub Staribus Common Port to read directly from these files, to test the assembly without any EEPROMs fitted. To do this edit instruments-common.xml to select the stub streams:
TxStream>org.lmn.fc.frameworks.starbase.plugins.observatory.ui.instruments.impl.staribuscontroller.dao.StaribusStubTxStream/TxStream>
RxStream>org.lmn.fc.frameworks.starbase.plugins.observatory.ui.instruments.impl.staribuscontroller.dao.StaribusStubRxStream/RxStream>
Don't forget to restore the original real streams when you are finished:
TxStream>org.lmn.fc.frameworks.starbase.plugins.observatory.ui.instruments.impl.staribuscontroller.dao.StaribusTxStream/TxStream>
RxStream>org.lmn.fc.frameworks.starbase.plugins.observatory.ui.instruments.impl.staribuscontroller.dao.StaribusRxStream/RxStream>
The DiscoveryTester can be used with a real Futurlec Controller, if you wish,
just use the real Streams as above.
