Starbase WebServer



Starbase has a simple embedded webserver, to allow direct publication of observations from any Instrument.

You may connect to the webserver's index page using any browser, by going to URL http://localhost:8080/index.html

Other Instruments may publish their results to this server, using commands like publishResults()

If you want to experiment, go to installation_path/webroot and modify the index.html file.

BEWARE! This 'tiny browser' only supports HTML v3.2, which isn't very clever!

For information about the webserver, see http://jetty.codehaus.org/jetty/

Note that the Starbase browser currently does not have a 'Back' button, so if you go off to Jetty the only way back is to stop and restart the server!
