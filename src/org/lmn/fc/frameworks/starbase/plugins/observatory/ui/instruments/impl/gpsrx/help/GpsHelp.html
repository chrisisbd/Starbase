Starbase GPS Receiver



The GPS Receiver interprets the NMEA 0183 standard protocol, used by most GPS receivers to transmit data.
NMEA output is EIA-422A but for most purposes you can consider it RS-232 compatible.
Use 4800 bps, 8 data bits, no parity and one stop bit (8N1).
NMEA 0183 sentences are all ASCII. Each sentence begins with a dollar sign ($)
and ends with a carriage return linefeed (CR LF).
Data are comma delimited. All commas must be included as they act as markers.
Some GPS receivers do not send some of the fields.
A checksum is optionally added (in a few cases it is mandatory).
Following the $ is the address field aaccc.
aa is the device id. Transmission of the device ID is usually optional.
'GP' is used to identify GPS data.
'ccc' is the sentence formatter, otherwise known as the sentence name.

e.g. The Garmin GPS35-PC unit produces the following NMEA sentences:

$GPGGA Global Positioning System Fix Data
$GPGSA GPS DOP and Active Satellites
$GPRMC Recommended minimum specific GPS/Transit data
$GPGSV GPS Satellites in view

To run the GPS Receiver, first select the GPS module.

Use the getInterpretedNMEA() command to see which NMEA sentences are currently supported by Starbase.
This information is stored in Starbase, NOT in the GPS receiver itself.
(If some sentences are not supplied, then it may still be possible to obtain some useful information.)

To obtain the latest position from the GPS Receiver, execute the getGpsFix() command.

To capture repeated fixes and produce a scatter plot of the results, execute the captureScatterPlot() command.
This command will calculate the centroid of the accumulated fixes, or just use the latest fix.
It is possible to choose to update the location of the Framework, the Observatory, or both.

Look in the InstrumentLog tab to see the detailed GPS information returned by the receiver.
Look in the EventLog tab to see if there are any warnings about missing or invalid data.
(It may be possible to configure your receiver to produce the required NMEA sentences.)

The SatelliteLocations tab will show all currently visible satellites
on an Elevation-Azimuth projection. It will also show the position of the Sun and other objects.
To clear the satellite tracks and start again, execute the reset() command.

Zoom In
Left_click zooms in
Shift-left_click zooms to the full extents of the display
Left_click in lower left corner (also) zooms to full extents

Zoom Out
Right_click zooms out
Shift-right_click zooms to the full extents of the display

Properties
Control-right_click looks for the nearest object and shows a properties box

Reserved
Control-left_click moves the cursor in tracking mode (not implemented yet)
