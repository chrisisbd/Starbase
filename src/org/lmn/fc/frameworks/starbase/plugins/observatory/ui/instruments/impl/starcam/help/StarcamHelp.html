Starbase Starcam


The Ethernet Webcam used to test the operation of Starcam was a D-Link DCS-900.
Note that this camera has a built-in webserver, so Starbase requests the image with HTTP.
If you use a different type of camera it may not be possible to import the image into Starbase.
Please ask for support on starbase@ukraa.com **before** purchasing your camera!
See: http://www.dlink.com/products/?pid=270
