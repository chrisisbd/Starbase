Starbase Starinet VLF

Uploading a new Controller Operating System

Go to the Manager Group tab, and execute the following command:

ObservatoryExplorer.Explorer.uploadOS()

You will need the latest Starinet-Nanode HEX file from the downloads at
http://www.ukraa.com/beta/nightly-builds/Starinet.
Remember to set the Controller into Bootloader mode before using this command.
Hold down the bootloader button, then press reset. Then release both buttons.
The LED should blink with a different pattern to show that the mode has changed.

Scanning for Starinet compatible devices on the local intranet

Go to the Manager Group tab, and select the ObservatoryExplorer.
Click on the Starinet Scanner tab, and enter the required range of IP addresses.
Any correctly working Starinet devices will respond with their host name.
In the current implementation of Starbase, you may need to manually edit
the IPAddress field in the Instrument XML file in order to use the discovered IP address,
and restart the Observatory. Work is in progress to simplify this procedure.
Note that this facility does not work in the public Internet,
in fact such scanning might be regarded as a criminal act!
