Starbase Terminal Emulator



This is a very simple Terminal Emulator, intended for debugging Instruments using RS232/485

The communication parameters are set in TerminalEmulator-properties.xml

Note that it is not possible to send just a CR:

User types LF (ctrl-J) Terminal sends CR (ctrl-M) followed by LF (ctrl-J)

User types CR (ctrl-M) Terminal sends CR (ctrl-M) followed by LF (ctrl-J)

i.e. they both produce the same output.
This is due to a limitation in Java,
which cannot read the CR key independently of the LF key.
It is unlikely that this limitation will cause problems,
but you need to be aware of it.
Windows and Linux handle CR/LF differently,
and this feature should make the differences less apparent.

If you find a device which does not function correctly, please let us know.
