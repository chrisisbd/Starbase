Metadata Editor Help


Warning!
Changing the DataType or the Units of the Metadata item may cause unpredictable problems!
For instance, if you changed a Latitude to a Longitude, or a floating point number to an Integer, then the effects may not be apparent until specific data values occur.

The Editor will check that the Value you have entered is valid for the selected DataType.
The Value field is limited to 1000 characters, Descriptions are limited to 100 characters.
The values of the DataTypes are checked using Regular Expressions (Regex), to ensure that it is not possible to save invalid data. (see: http://en.wikipedia.org/wiki/Regular_expression)

Note that any changes you make are written to the Metadata associated with the current data set.
In order to save your changes, you must export the RawData or ProcessedData (preferably as Stardata XML) before capturing or importing any other data. In other words, the Metadata does not exist independently of the observational data.

All Metadata changes are recorded in the EventLog, and in the ObservatoryLog, if it is running.

For information about ISO standards, see: http://en.wikipedia.org/wiki/ISO_8601
