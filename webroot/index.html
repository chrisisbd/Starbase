This is the default index page of the Starbase embedded web server, powered by Jetty.

You may connect to this page using any browser, by going to URL http://localhost:8080/index.html

Other Instruments may publish their results to this server, using commands like publishResults()

If you want to experiment, go to installation_path/webroot and modify this file.

BEWARE! This 'tiny browser' only supports HTML v3.2, which isn't very clever!

See http://jetty.codehaus.org/jetty/

Note that the Starbase browser currently does not have a 'Back' button, so if you go off to Jetty the only way back is to stop and restart the server!
